# Pharmacy Checkout System (Java)

## Description
A console-based checkout system inspired by my experience working in a retail pharmacy.
The program simulates a simple checkout flow using predefined items, allowing users to
add items to a cart and calculate the total price.

## Features
- Predefined pharmacy items
- Add items to cart
- View cart contents
- Calculate total price
- Input validation

## Concepts Demonstrated
- Java classes and objects
- Arrays and ArrayLists
- Loops and conditionals
- Methods and program flow
- Basic object-oriented design

## How to Run
1. Clone the repository
2. Compile and run `Main.java`
